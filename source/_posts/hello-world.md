## 快速开始

### 创建新文章

``` bash
$ hexo new "My New Post"
```

更多信息：[写作](https://hexo.io/docs/writing.html)

### 运行服务器

``` bash
$ hexo server
```

更多信息：[服务器](https://hexo.io/docs/server.html)

### 生成静态文件

``` bash
$ hexo generate
```

更多信息：[生成](https://hexo.io/docs/generating.html)

### 部署到远程网站

``` bash
$ hexo deploy
```

更多信息：[一键部署](https://hexo.io/docs/one-command-deployment.html)
